# Bundled fonts

AnimaForge bundles four open-source Google Fonts so the UI renders identically
on any machine (loaded at startup via `QFontDatabase`, see `utils/fonts.py`).

| File | Family | License | Source |
|------|--------|---------|--------|
| `PirataOne-Regular.ttf` | Pirata One | SIL Open Font License 1.1 | https://fonts.google.com/specimen/Pirata+One |
| `PermanentMarker-Regular.ttf` | Permanent Marker | Apache License 2.0 | https://fonts.google.com/specimen/Permanent+Marker |
| `SpecialElite-Regular.ttf` | Special Elite | Apache License 2.0 | https://fonts.google.com/specimen/Special+Elite |
| `CrimsonPro-Variable.ttf` | Crimson Pro (variable) | SIL Open Font License 1.1 | https://fonts.google.com/specimen/Crimson+Pro |

Both the SIL OFL 1.1 and the Apache License 2.0 permit bundling and
redistribution with an application. Full license texts are available at the
source links above and in the upstream `google/fonts` repository.
